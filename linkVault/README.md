# linkVault
Este é um projeto feito com o intuito de estudar o uso de ferramentas de **Observabilidade** e **métricas**, pra ser mais especifico, o **Actuator**, **Grafana** e **Prometheus**.
Inicialmente esse projeto será apenas um vault onde poderá ser guardados links importantes, futuramente planeja-se implementar um módulo de **URL shortener**.

# Estrutura
O projeto está organizado em uma estrutura **Monolítica Modular** por questões de organização e simplicidade. na primeira versão estará presente o módulo principal de operações envolvendo links.

# Ferramentas:
Linguagens: Java, Typescript, Kotlin
Frameworks: Spring Boot 4, Angular 20, JatPack Compose
Ambiente: Docker, Prometheus, Grafana, Linux
